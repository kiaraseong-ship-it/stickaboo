How to use:

  1. Create an Excel file with the following columns: First Name, Last Name, Size, Theme, Font Color
  2. Valid sizes: small, medium, large, sml-mix, ml-mix
  3. Valid themes: undertheocean, animal, dino, unicorn, space, cuteicon, nameonly
  4. Valid font colors: Black, White, Navy, Blue, Green, Pink, Brown (or hex codes like #000000)
  5. Upload your Excel file and click "Process Excel"
